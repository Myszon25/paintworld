The R Function

A recursive tetration hierarchy for extremely large numbers

1. Definition of the first level

The first level of the R function is:

R₁(3) = 3^^3

The symbol ^^ means tetration. It creates a power tower.

3^^3 = 3(33)

R₁(3) = 7,625,597,484,987

2. Recursive growth

Each new R level uses the previous level recursively.

R₂(3) = R₁(R₁(3))

The function applies itself again using the previous result as input.

R₃(3) = R₂(R₂(3))

Every increase of the index creates a much faster growing number.

3. General notation

Rₓ(n)
  • x = number of recursive layers
  • n = starting value

Examples:

R₁ = tetration
R₂ = recursive tetration
R₃ = recursion of recursion

4. Sigma Function (Σ)

Sigma is a function that creates the level of recursion.

Σ(n) = n^^n

The result of Σ becomes the index of R.

Example 1:

Σ(2)=2^^2
2^^2=4
RΣ(2)=R₄(2)

Example 2:

Σ(3)=3^^3
Σ(3)=7,625,597,484,987
RΣ(3)=R₇,625,597,484,987(3)

5. Meaning

RΣ(n) means:

RΣ(n)=R₍Σ(n)₎(n)

First calculate Σ(n), then use that number as the recursion level of R.

The recursion depth itself becomes an enormous number.

6. Concept

The R function is a custom fast-growing hierarchy inspired by googology. It is designed to describe extremely large numbers that cannot be written normally.

  • Exponentiation
  • Tetration
  • Recursive tetration
  • Higher recursive levels

The purpose is not calculating the exact value, but describing the speed of growth.
